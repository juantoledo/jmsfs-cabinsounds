# Usage

Just download the latest bundle zip version from  [Releases]( https://github.com/juantoledo/jmsfs-cabinsounds/releases) page, unzip it, execute jmsfs-cabinsounds.exe and you're ready to go.


# If you need to add your own audio Files 

Just replicate the following structure under the **audiofiles** directory 


* Airline Name 1
  * 1.Sound Category 1
    * 1.Soundfile1.wav
    * 2.Soundfile2.wav
    * n.Soundfilen.wav  
  
  * 2.Sound Category 2
    * 1.Soundfile1.wav
    * 2.Soundfile2.wav
    * n.Soundfilen.wav 
   

* Airline Name 2
  * 1.Sound Category 1
    * 1.Soundfile1.wav
    * 2.Soundfile2.wav
    * n.Soundfilen.wav  
  
  * 2.Sound Category 2
    * 1.Soundfile1.wav
    * 2.Soundfile2.wav
    * n.Soundfilen.wav
